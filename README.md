# 📊 Expense Tracker

A simple **CLI-based expense tracker** using Python and SQLite.

## Features
✅ Add expenses  
✅ View all expenses  
✅ Monthly summary by category  
✅ Delete an expense  
✅ Export to CSV  

## 📌 How to Run
```sh

cd expense-tracker
python3 -m venv venv
